Create a lib directory (if not already present) on your Pico W via Thonny
Open the secrets.py file and insert your ssid and password between the quotes
Example:
  ssid = "YOUR_WIFI_SSID" to ssid ="daves_router"
  password = "YOUR_WIFI_PASSWORD" to password = "OU812"
Once this completed save the file. navigate to the lib folder on your pico and upload your secrets.py file
Run the astronauts_and_iss_location.py
Note: this may work for ESP32 micros but I have not tried it as the micropython should be the same.
